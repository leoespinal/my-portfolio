Mem Sim

An iOS application that serves as a memory management simulator that can be used by computer science students who are taking an Operating Systems course and want to simulate how the paging memory management scheme in an operating system works. Fully developed in Swift 3. Uses Apple's UIKit Framework objects and protocols such as UISplitView, UITableViewController, UITableViewDelegate, and UITableViewDataSource.
